# Mithril-Dos

A basic To-Do app built with [Mithril](https://mithril.js.org/), [Vite](https://vitejs.dev/) and TypeScript.


## Get Started

```bash
# Clone project
git clone git@github.com:roymckenzie/mithril-dos.git

# Fetch dependencies
cd mithril-dos
npm install

# Start Vite
npm run dev
```
